# Network Automation Labs

Hands-on Python automation labs for network engineers.

This repository contains real-world, CLI-focused automation labs designed for
network engineers who want to automate day-to-day operations using Python.

## What You Will Learn
- Automating network tasks using Python
- Working with real network devices (Cisco IOS)
- Writing safe, repeatable automation scripts
- Thinking like a Network Automation Engineer

## Lab Structure
Each lab includes:
- A real-world problem statement
- Lab topology (EVE-NG / GNS3)
- Step-by-step instructions
- Python scripts
- Expected output
- Common mistakes and fixes
