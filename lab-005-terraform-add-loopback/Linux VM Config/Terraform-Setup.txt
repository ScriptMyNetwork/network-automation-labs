## Terraform Installation (Linux)

The following steps install Terraform on a Linux-based automation host. This method uses the official HashiCorp release binaries and avoids distribution-specific package inconsistencies.

---

### 1. Update System Packages

sudo apt update

---

### 2. Install Required Utilities

sudo apt install -y wget unzip

---

### 3. Download Terraform Binary

wget https://releases.hashicorp.com/terraform/1.6.6/terraform_1.6.6_linux_amd64.zip

---

### 4. Extract the Binary

unzip terraform_1.6.6_linux_amd64.zip

---

### 5. Move Terraform to PATH

sudo mv terraform /usr/local/bin/

---

### 6. Verify Installation

terraform version

The output should display the installed Terraform version.

---

### Notes

- Installing Terraform manually ensures version control and reproducibility.
- This approach aligns with enterprise automation and CI/CD environments.
- Terraform upgrades should be performed deliberately and tested before use.
